# leancontext.dev

The Lean Context Thesis — How Humans and AI Agents Make Each Other Smarter.

A framework for structuring AI agent context as a lean system: constitution, manifest, and supply.

## Building

```bash
hugo server --buildDrafts
```

## Deploying

```bash
hugo --minify
```

The `public/` directory contains the static site ready for deployment (GitHub Pages, Netlify, Cloudflare Pages, etc).
