Key concepts

Constitution

Always loaded. Hard stops only. If the agent misses it once, the damage is irreversible. Everything else lives somewhere else.

Manifest

A routing table. The agent sees what context exists without holding it. Matches the task, pulls the document. Shadow board for the workbench.

Supply

Pulled on demand. One topic per document. Only invisible knowledge — the things the agent could never discover from the code alone.

Andon Rule

Stop when uncertain. Surface what is missing. Help fill the gap. The system gets smarter every time the agent stops instead of guessing.

Session Discipline

One task. One session. Clean workbench. Problem solved or gap filled — the session ends. Context lives in the repo, not the conversation.

Context Compaction

The human maintains the system between sessions. Demote, prune, validate. The agent never needs to compress within a session because the system was built so it wouldn't have to.
