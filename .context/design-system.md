# Design System

## Colors

- Background: `#0d1117` (--bg)
- Surface: `#161b22` (--bg-secondary)
- Text: `#c9d1d9` (--fg)
- Muted text: `#8b949e` (--fg-muted)
- Headings: `#e6edf3` (--fg-heading)
- Accent (links): `#58a6ff` (--accent)
- Borders: `#21262d` (--border)

## Typography

- Body: Georgia, serif (--font-body), 18px base
- Headings: System sans-serif stack (--font-heading)
- Code: SFMono/Consolas monospace stack (--font-mono)

## Layout

- Max content width: 38em (--measure)
- Article padding: 4rem top, 6rem bottom, 1.5rem sides
- Mobile breakpoint: 640px (font drops to 16px)

## Principles

- Dark background throughout. No light mode.
- Minimal decoration. No images, icons, or illustrations on content pages.
- The text is the interface. Typography and spacing do all the work.
- No JavaScript on content pages. The site is fully static HTML+CSS.
- Navigation is monospace, small, top-aligned. It does not compete with content.
